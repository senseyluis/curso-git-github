# Curso Git e GitHub

Curso para iniciantes aprenderem a trabalhar com versionaento de código e repositórios remotos com GitHub.

## Fluxo de trabalho Git local

1.  git checkout -b <nova branch>
2.  cria ou atualiza arquivos
3.  git status
4.  git add _arquivos_
5.  git status
6.  git commit -m "Minha mensagem"
7.  git checkout main
8.  git merge nova branch
